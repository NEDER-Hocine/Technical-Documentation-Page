Introduction

C is a general-purpose, procedural programming language developed by Dennis Ritchie in 1972 at Bell Labs. Known for its efficiency and flexibility, C is widely used in system programming, embedded systems, and application development. It serves as the foundation for many other programming languages like C++, Java, and Python.

What You Should Already Know

Before starting with C, it is recommended that you have:

  • Basic Programming Concepts: Understanding variables, loops, and functions.
  • Problem-Solving Skills: Ability to break down tasks into smaller steps.
  • Basic Knowledge of Computers: Familiarity with compilers and text editors.
Why Learn C?
  • Performance: C provides low-level access to memory and efficient execution.
  • Portability: Programs written in C can run on various platforms with minimal changes.
  • Foundation: Many modern languages derive their syntax and concepts from C.
  • Versatility: C is used for system programming, embedded systems, game development, and more.
Hello World Example

A simple program to print "Hello, World!" in C:

#include < stdio.h>

int main() {
printf("Hello, World!\n");
return 0;
}

Structure of a C Program

A C program typically includes the following components:

  1. Preprocessor Directives: Statements starting with #, such as #include and #define.
  2. Main Function: Entry point of the program, defined as int main().
  3. Statements and Expressions: Executable instructions.
  4. Return Statement: Indicates the end of the program.

Example:

#include < stdio.h>

int main() {
int number = 10; // Variable declaration
printf("Number is: %d\n", number);
return 0;
}

Data Types

C provides various data types to define variables and constants:

  • Basic Types: int, float, double, char.
  • Derived Types: array, pointer, struct, union.
  • Void: Represents no type.

Example of declaring variables:

int age = 25;
float height = 5.9;
char grade = 'A';

Control Structures

C supports control flow constructs to manage program execution:

  1. Conditional Statements: if, else, switch.
  2. Loops: for, while, do-while.
  3. Jump Statements: break, continue, goto.

Example of a loop:

for (int i = 0; i < 5; i++) {
printf("Iteration %d\n", i);
}

Functions

Functions in C allow modular programming by encapsulating reusable code.

  • Syntax:

    return_type function_name
    (parameters) {
    // function body
    }

  • Example:

    int add(int a, int b) {
    return a + b;
    }

  • Usage:

    int result = add(5, 3);
    printf("Result: %d\n", result);

Arrays

Arrays store multiple elements of the same type in contiguous memory locations.

Example:

int numbers[5] = {1, 2, 3, 4, 5};
for (int i = 0; i < 5; i++) {
printf("%d\n", numbers[i]);
}

Structures

Structures allow grouping of different data types under one name.

Example:

struct Student {
int id;
char name[50];
float grade;
};

struct Student s1 = {1, "Alice", 92.5};
printf("Name: %s, Grade: %.2f\n", s1.name, s1.grade);

File Handling

C provides functions for reading from and writing to files.

Example:

FILE *fptr = fopen("example.txt", "w");
fprintf(fptr, "Hello, File!");
fclose(fptr);

Memory Management

Dynamic memory allocation is managed using functions like malloc, calloc, realloc, and free.

Example:

int *ptr = (int *)malloc(5 * sizeof(int));
if (ptr == NULL) {
printf("Memory not allocated.\n");
} else {
// Use memory
free(ptr); // Free memory
}

Best Practices
  • Always initialize variables.
  • Use comments to explain your code.
  • Check for null pointers before accessing memory.
  • Use consistent indentation for better readability.
Conclusion

C remains a vital language in the programming world due to its power and efficiency. Mastering C can build a strong foundation for learning advanced programming languages and concepts.
